# SkillBridge – OpenAI-Powered Employability Analysis

SkillBridge is a full-stack web application that analyzes user resumes and uses the OpenAI API to extract skills, perform skill gap analysis against a target job role, and (upcoming) recommend learning paths.

## 🚀 Features (Completed)
- Resume upload (PDF/DOCX)
- Resume text extraction
- OpenAI-based skill extraction (technical, soft skills, tools)
- OpenAI-based skill gap analysis (matched skills, missing skills, readiness score)
- React frontend + Flask backend integration
- Secure API key handling using environment variables

## 🧱 Tech Stack
- Backend: Python (Flask)
- Frontend: React (Vite)
- AI: OpenAI API
